##  状态机

状态机的基本定义：状态机是有限状态自动机的简称，是现实事物运行规则抽线而成的一个数学模型

状态机里的4个概念
- State 状态 ： 一个状态机必须要包含两个状态，每次都是从 开始态开始。
- Event 事件 ： 执行某个操作的触发条件
- Action 动作： 事件发生以后要执行的动作
- Transition 转换： 状态的变化


在这道题里， 程序在每个时刻有一个状态 S ，每次从字符串中读取一个字串 C ，并根据字符 C 进行相应的动作，然后转移到下一个动作S`.
